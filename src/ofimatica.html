Plan de Estudios - Técnico en Ofimática

MÓDULO I: Introducción a la Ofimática

Este módulo introduce al estudiante en el mundo de las aplicaciones ofimáticas y el entorno de trabajo en la oficina.

  • Submódulo 1: Fundamentos de Ofimática
    60 horas

    En este submódulo se aprenderán los conceptos básicos de las herramientas ofimáticas, incluyendo el uso de sistemas operativos, dispositivos y aplicaciones de oficina.

  • Submódulo 2: Introducción al uso de Microsoft Office
    80 horas

    Este submódulo cubre el manejo de las principales aplicaciones de Microsoft Office, como Word, Excel y PowerPoint.

Total: 140 horas

MÓDULO II: Procesador de Texto (Microsoft Word)

En este módulo se profundiza en el uso del procesador de texto para la creación de documentos, cartas, informes y otros archivos de texto profesionales.

  • Submódulo 1: Creación de documentos
    80 horas

    Los estudiantes aprenderán a crear documentos de texto utilizando herramientas de formato, plantillas y tablas, así como a insertar imágenes, enlaces y objetos.

  • Submódulo 2: Formato avanzado de documentos
    80 horas

    Este submódulo enseña a usar funcionalidades avanzadas de Word, como el uso de estilos, referencias bibliográficas, tablas de contenido y macros.

Total: 160 horas

MÓDULO III: Hojas de Cálculo (Microsoft Excel)

Este módulo cubre el uso de Excel para la creación de hojas de cálculo, análisis de datos y generación de informes financieros.

  • Submódulo 1: Fundamentos de Excel
    80 horas

    Los estudiantes aprenderán a crear y dar formato a hojas de cálculo, además de realizar operaciones matemáticas básicas y uso de funciones simples.

  • Submódulo 2: Análisis avanzado de datos
    100 horas

    Este submódulo enseña el uso de herramientas avanzadas de Excel como gráficos, tablas dinámicas, validación de datos y el análisis de grandes volúmenes de información.

Total: 180 horas

MÓDULO IV: Presentaciones Multimedia (Microsoft PowerPoint)

Los estudiantes aprenderán a crear presentaciones efectivas usando PowerPoint, incluyendo el diseño y la incorporación de elementos multimedia.

  • Submódulo 1: Diseño de presentaciones
    60 horas

    En este submódulo se aprenderán las herramientas básicas para crear presentaciones, como el uso de plantillas, diseño de diapositivas y organización de contenido.

  • Submódulo 2: Presentaciones multimedia interactivas
    80 horas

    Este submódulo enseña a integrar elementos multimedia como imágenes, audios, videos y animaciones dentro de las presentaciones para hacerlas más dinámicas.

Total: 140 horas

MÓDULO V: Gestión de Archivos y Correo Electrónico

Este módulo se centra en la organización, almacenamiento y gestión de archivos en el entorno digital, además de la gestión del correo electrónico para fines profesionales.

  • Submódulo 1: Gestión de archivos digitales
    60 horas

    Aprenderás a organizar, almacenar y recuperar documentos digitales de manera eficiente usando carpetas, servidores y servicios en la nube.

  • Submódulo 2: Uso profesional del correo electrónico
    80 horas

    Este submódulo cubre el uso del correo electrónico en el entorno profesional, incluyendo la creación de firmas electrónicas, la gestión de bandejas de entrada y la seguridad.

Total: 140 horas
