Plan de Estudios - Técnico en Programación

MÓDULO I: Fundamentos de Programación

Este módulo proporciona una base sólida en los principios fundamentales de la programación, algoritmos y estructuras de datos.

  • Submódulo 1: Introducción a la programación
    80 horas

    Aprenderás los conceptos básicos de programación, como variables, operadores, estructuras de control (condicionales, bucles) y cómo implementar soluciones simples.

  • Submódulo 2: Algoritmos y estructuras de datos
    120 horas

    Te adentrarás en la creación de algoritmos eficientes y el manejo de estructuras de datos como arreglos, listas enlazadas, pilas, colas, entre otros.

  • Submódulo 3: Lenguajes de programación (C, Python)
    160 horas

    Estudiarás los fundamentos de dos lenguajes de programación populares: C para entender el control de memoria y Python para aplicaciones rápidas y fáciles de usar.

Total: 360 horas

MÓDULO II: Desarrollo de Aplicaciones Web

Este módulo se centra en el diseño, desarrollo y programación de sitios web interactivos, utilizando tecnologías modernas.

  • Submódulo 1: HTML y CSS para la construcción de sitios web
    100 horas

    Aprenderás a crear la estructura básica de un sitio web con HTML y a estilizarlo utilizando CSS para hacerlo atractivo y funcional.

  • Submódulo 2: JavaScript y la interacción con el usuario
    120 horas

    Te introducirás a JavaScript para hacer que tu sitio web sea interactivo, incluyendo la manipulación del DOM y el uso de eventos.

  • Submódulo 3: Desarrollo de aplicaciones con frameworks (React)
    140 horas

    Aprenderás a utilizar React, un framework popular para el desarrollo de aplicaciones web dinámicas y escalables.

Total: 360 horas

MÓDULO III: Bases de Datos

En este módulo aprenderás a diseñar, gestionar y consultar bases de datos de manera eficiente para aplicaciones web y móviles.

  • Submódulo 1: Introducción a bases de datos relacionales
    100 horas

    Te familiarizarás con las bases de datos relacionales y el modelo de tablas, claves y relaciones.

  • Submódulo 2: SQL y diseño de bases de datos
    120 horas

    Estudiarás el lenguaje SQL para crear, gestionar y consultar bases de datos, además de diseñar esquemas de bases de datos eficientes.

  • Submódulo 3: Administración de bases de datos
    140 horas

    Aprenderás las mejores prácticas para administrar bases de datos, realizar copias de seguridad, optimizar consultas y mantener la seguridad.

Total: 360 horas

MÓDULO IV: Programación de Aplicaciones Móviles

En este módulo, te enfocarás en la creación de aplicaciones móviles tanto para Android como para iOS utilizando las herramientas y lenguajes apropiados.

  • Submódulo 1: Desarrollo de aplicaciones Android con Java
    120 horas

    Desarrollarás aplicaciones móviles para dispositivos Android utilizando Java, desde la interfaz de usuario hasta la lógica de negocio.

  • Submódulo 2: Desarrollo de aplicaciones iOS con Swift
    120 horas

    Estudiarás Swift para crear aplicaciones para dispositivos iOS, utilizando las mejores prácticas en diseño y programación.

Total: 240 horas

MÓDULO V: Desarrollo de Software Avanzado

Este módulo cubre técnicas avanzadas para la creación de aplicaciones de alto rendimiento y escalabilidad, incluyendo el desarrollo en la nube y microservicios.

  • Submódulo 1: Desarrollo de aplicaciones en la nube
    140 horas

    Aprenderás a desarrollar aplicaciones escalables y resilientes en plataformas en la nube como AWS, Google Cloud y Azure.

  • Submódulo 2: Integración de APIs y microservicios
    120 horas

    Te adentrarás en la integración de APIs externas y el desarrollo de aplicaciones basadas en microservicios para una mejor modularidad y escalabilidad.

Total: 260 horas
